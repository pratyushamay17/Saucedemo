# SauceDemo — BDD Playwright Tests

A small Playwright + Cucumber (Cucumber-js) test suite for automating a purchase flow on SauceDemo.

## Contents
- tests/features — .feature files (Gherkin)
- tests/steps — Cucumber step definitions (TypeScript)
- tests/page-objects — Page object classes (TypeScript)
- cucumber.cjs / cucumber.js — Cucumber config
- package.json — scripts & dependencies

## Prerequisites
- Node.js 18+ (Windows)
- npm
- VS Code (recommended)

## Clone the SauceDemo project and open terminal
  - npm install

## Run tests with below command
 - npm run bdd
